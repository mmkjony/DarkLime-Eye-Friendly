DarkLime - an Eye Friendly Color Scheme for Sublime Text.
===================================
DarkLime is an eye friendly dark color scheme for Sublime Text, that makes your codes easy for your eyes to read & write.

-----------
![HTML Code Preview](http://i.imgur.com/eQHXnjm.png)

Installation
----------------
 * Use [Sublime Package Control](http://wbond.net/sublime_packages/package_control) and search for DarkLime.
 * Alternative: Just download and copy to package directory.. Enjoy!
